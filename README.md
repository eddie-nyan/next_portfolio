# Personal Portfolio Website

A modern, responsive portfolio website built with Next.js and TailwindCSS, showcasing my work as a Full Stack Developer.

## 🚀 Features

- Modern UI with smooth animations using Framer Motion
- Responsive design that works on all devices
- Dark theme optimized for readability
- Interactive components and transitions
- SEO optimized
- Fast loading with Next.js optimizations

## 🛠 Tech Stack

- **Framework:** Next.js 14
- **Styling:** TailwindCSS
- **Animations:** Framer Motion
- **Icons:** Heroicons
- **Deployment:** Vercel
- **Analytics:** Vercel Analytics

## 🏃‍♂️ Running Locally

1. Clone the repository:
```bash
git clone https://github.com/eddie-nyan/next_portfolio.git
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📁 Project Structure

```
├── public/          # Static assets
├── src/
│   ├── app/        # App router pages
│   ├── components/ # React components
│   └── styles/     # Global styles
└── tailwind.config.js
```

## 📝 Environment Variables

Create a `.env.local` file in the root directory:

```
NEXT_PUBLIC_SITE_URL=your-site-url
```

## 📦 Build & Deploy

To build the project:

```bash
npm run build
# or
yarn build
```

Deploy to Vercel:

```bash
vercel
```
